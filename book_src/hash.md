# 雜湊值工具

雜湊值（Hash）是資料的「數位指紋」。任何對原始資料的微小修改都會使雜湊值產生巨大的改變。本工具主要用於驗證檔案是否被篡改或下載是否完整。

## 支援演算法
* **MD5**：128-bit，速度極快，因有碰撞風險，僅建議用於一般的檔案完整性比對。
* **CRC32**：32-bit，常見於檔案校驗與 ZIP/壓縮格式的整合性檢查，適合快速檢查資料是否被修改。
* **SHA-1**：160-bit，因安全性降低，建議用於舊系統相容或非安全性校驗。
* **SHA-256**：256-bit，目前最推薦的安全雜湊標準。
* **SHA3-256**：最新的 Keccak-256 標準，提供極佳的防碰撞安全性。

## 操作步驟

### 1. 計算雜湊值
1. 切換至 **Hash Matrix** 分頁。
2. 選擇 **Target Type**：
   * **File Path**：計算整個檔案的雜湊值（支援大檔案串流，不佔用大量記憶體）。
   * **Plain Text**：計算輸入框內文字的雜湊值。
3. 選擇要使用的 **Algorithm (演算法)**（推薦 `SHA-256`）。
4. 在 **Source Input** 輸入文字或選擇檔案路徑。
5. 點擊 **Compute Hash** 進行計算，結果將顯示在 **Hash Result** 中。
6. 點擊 **Copy Hash** 複製雜湊值。

### 2. 完整性對比 (Compare)
1. 計算出檔案的雜湊值後。
2. 點擊 **Compare...** 按鈕。
3. 彈出對話框中，您可以貼上官方提供的校驗碼（或選擇一個已有的 `.sha256` 雜湊校驗檔案）。
4. 軟體會自動執行比對，若一致會彈出 `Hash Match!` 成功提示，不一致則會發出警告，提示檔案可能已損毀或被修改。
