# 前言

歡迎使用 **Kitana Cryptool** 官方使用手冊！

Kitana Cryptool 是一款基於 **Rust** 語言與 **FLTK** 圖形框架開發的輕量化、多功能安全工具箱。本工具旨在提供直覺且高效的加密、雜湊與金鑰管理方案，保護您的敏感性資料與檔案安全。

### 本書架構
* **AES 檔案加解密**：使用軍規級的 AES-256-CBC 串流技術對任何大小的檔案進行加解密。
* **RSA 混合加/解密與簽章**：結合 RSA 與 AES 優點，實現安全、無長度限制的混合加解密與數位簽章驗證。
* **RSA 金鑰產生器**：產生不同長度的 RSA 金鑰對，並可匯出為標準 PEM 格式及 OpenSSH 公鑰。
* **安全密碼產生器**：產生高度隨機的隨機密碼，並配備密碼強度指示器。
* **雜湊值工具**：提供多種主流雜湊演算法，協助您快速驗證檔案的完整性。

### 設計哲學
* **安全至上**：所有密碼學操作均由經過社群嚴格審查的 RustCrypto 系列庫完成。
* **無快取殘留**：工具不會將您的明文、密碼或私鑰上傳或儲存至本機磁碟的任何快取區。
* **輕量高效**：無任何 Electron 等肥大框架，啟動時間低於 0.1 秒，記憶體佔用極低。
