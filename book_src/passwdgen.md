# 安全密碼產生器

重複使用簡單的密碼是帳戶被盜的最主要原因。Kitana 內建的安全密碼產生器能協助您在本機生成足夠複雜且隨機的強密碼。

## 技術規格
* **真隨機來源**：使用 `rand::rngs::OsRng` 呼叫作業系統底層核心的安全隨機數來源（如 Windows 的 CryptGenRandom API），確保生成的密碼無法被預測。
* **字元集選擇**：
  - **Uppercase**: `A-Z`
  - **Lowercase**: `a-z`
  - **Digits**: `0-9`
  - **Symbols**: 特殊符號集（如 `!@#$%^&*()_+-=` 等）
* **密碼強度評估**：內建基於熵值（Entropy Bits）的即時強度檢測條，當密碼字元多樣性與長度足夠時，強度條會變綠。

## 操作步驟
1. 切換至 **Password Generator** 分頁。
2. 拉動 **Length** 滑桿設定所需的密碼長度（推薦 16 字元以上）。
3. 勾選想要包含的字元種類。
4. 點擊 **Generate Password**。
5. 生成的密碼會顯示在輸入框中，下方的強度條會顯示此密碼的強度等級。
6. 點擊 **Copy Password** 將密碼一鍵複製到剪貼簿中。
7. 使用完畢後，點擊 **Clear** 清除輸入框，防止密碼殘留在螢幕上。
